AION Chat is an utility app to read AION Chat.log in real time.

![](https://i.imgur.com/tyhB0rJ.png)

## Features ##

* Real time update
* Chat channels
* Chat triggers
* Ban chat words
* Easy copy paste
* Sound / Tray notification
* Link parsing
